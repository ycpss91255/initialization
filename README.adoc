== 安裝

link:./README.adoc[English]

[NOTE]
====
需要相應地轉換以下內容。

* `<README_PATH>` 是 *README.adoc* 檔案的所在目錄。
====

[source, shell]
----
# 安裝 docker 和 nvidia-container-toolkit
cd <this directory>
chmod +x ./install.sh
./install.sh

# 驗證安裝
docker -v
nvidia-container-toolkit -version
----
